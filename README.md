## Project Name

Tiles Gallery

## Purpose

This is a modern tiles gallery web application built with **Next.js**.  
It allows users to browse, search, and view different tile designs in a clean and responsive UI.

## Live URL

https://tiles-gallary-nextjs.vercel.app

## Key Features

- Search tiles by title
- Responsive image gallery
- Fast performance using Next.js
- Clean and modern UI design
- Fully mobile responsive
- Authentication system (Google login supported)
- Server + Client component architecture

## NPM Packages Used

- next
- react
- react-dom
- tailwindcss
- @heroui/react
